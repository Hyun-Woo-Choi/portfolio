# Career Outcomes Portfolio

A clean, responsive Next.js portfolio template focused on career outcomes.

## Features
- Gentle black/white modern UI (simple, credible tone)
- Who you are + interests section
- Project cards by company and project name
- Click-to-view project outcomes with text and optional image
- LinkedIn + email contact links
- Responsive layout for desktop/tablet/mobile

## Run locally
```bash
npm install
npm run dev
```

## Deploy
This project is ready for Vercel:
1. Push to GitHub
2. Import repository in Vercel
3. Deploy (default Next.js settings work)

## Customize
Update profile and project content in `app/data.js`.
